# 知乎 Plus
给知乎用户的 Alfred Workflow，支持 Alfred 2 / 3。
灵感源自 [KJlmfe](https://github.com/KJlmfe/Alfred-workflows/)。

## Feature - 功能
- 搜索知乎问题、文章、话题或用户，指令：zh
- 浏览最新知乎日报，指令：zhd
- 更多功能添加中
