# 知乎Plus

![zhihu-plus-logo](https://raw.githubusercontent.com/Astraeux/zhihu-plus-alfred-workflow/master/src/icon.png)

给知乎用户的 Alfred Workflow，支持 Alfred 2 / 3。



## Feature

- 搜索知乎问题、文章、话题或用户，指令：zh
- 浏览最新知乎日报，指令：zhd
- 更多功能添加中...
